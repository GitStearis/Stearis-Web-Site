Привет, путник! Мой никнейм - Stearis, я являюсь студентом-программистом первого курса. Главная идея этого сайта - быть площадкой для моих первых экспериментов в веб-разработке. Большинство изображений на моем сайте созданы/сверстаны/найдены мною.

При развитии этой страницы я приследую следующие цели:

  • Тренировка навыков сайтостроения
  • Создание собственного одностраничного сайта
  • Знакомство с системой доменов и хостингов

Сайт написан на HTML с использованием CSS. На сайте присутствует кнопка перемещения в начало страницы, созданная с помощью языка JavaScript. Разработка велась параллельно с выполнением лабораторных заданий, особенности, которые меня интересовали, нашли отражение на моем сайте.

<video>

Это видео проигрывается с использованием новых возможностей HTML. До HTML5 не существовало стандарта для отображения видео на веб-странице. Ранее видео могли проигрываться только с использованием плагинов (например flash). Тэг <video> открыл возможность прикреплять видеофайлы в стандартной манере. У этого способа есть существенный недостаток - ограниченное количество поддерживаемых кодеков.

<audio>

С введением технологии HTML5 у браузеров появилась замечательная возможность - воспроизводить аудиофайлы стандартным методом. Поддерживаемыми форматами являются MP3, WAV и OGG. HTML5 определяет новые DOM-методы и свойства для элемента <audio>.

<canvas>

В HTML5 определен элемент <canvas> как «растровый холст, который может быть использован для отображения диаграмм, игровой графики или изображений на лету». Холст это прямоугольная область на вашей странице, где с помощью JavaScript можно рисовать что пожелаете.






Geolocation

Геолокация - способ обнаружения географического положения устройства, подключенного к Интернету. В HTML5 определены новые методы для JS, позволяющие определять широту и долготу, откуда совершался запрос.
